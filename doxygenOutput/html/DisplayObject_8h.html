Other objects will inherit this class and override the draw() function.

When designing other objects.... Make a class in C++ that inherits this class, but overriding the draw() function. Inside you can use whatever OpenGL stuff you want.

You can call DEBUG_DISP() if you want to add the command to cout for debugging. Only active if DEBUG_DISP_CMD is defined. You will want to change DEBUG_DISP_CMD to a custom one per obejct. Eg. DEBUG_ORANGE_CMD for the orange object.
